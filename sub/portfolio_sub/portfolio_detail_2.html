수제마카롱 제조 D사

최상의 퀄리티를 자부하는 수제 마카롱 생산 업체!

최고의 맛을 위한 균일한 계량시스템, 엄선된 재료의 보관을 위한 온습도 관리시스템, 실시간모니터링으로 최고의 품질을 유지하는 시스템

프로젝트 개요

서비스 설명
온라인 쇼핑몰과 MES 프로그램을 연동하여 제품의 생산과 출고를 하나의 통합 시스템으로 관리하기 위한 프로그램
사용 기술
C#, MSSQL, Mobile(Android)
주요 기능
MES 구축
쇼핑몰 API 연동
실시간 설비 모니터링 프로그램
자동계량 프로그램

요구사항

  • 쇼핑몰 API 연동 모업체에서 사용중인 쇼핑몰의 정보를 MES 프로그램에 융합시켜 통합관리
  • 실시간 설비 모니터링 각 작업위치별 온도 및 습도정보를 실시간으로 모니터링
  • 자동계량 제품 생산을 위한 원재료 투입량을 전자저울로 자동계량

Our Solution

As-is

  • ✔ 생산, 입고, 출고 둥 기존 수기 작성
  • ✔ 현재 쇼핑몰과 ERP 프로그램을 사용하여 중복 업무 발생
  • ✔ 식품업체 특성상 작업장소별 온, 습도가 중요하나 파악 어려움
  • ✔ 원재료의 투입량 및 재고를 수기 관리

To-be

  • ✔ MES 구축
  • ✔ 쇼핑몰 API를 통해 MES 프로그램에서 통합관리
  • ✔ 온, 습도센서 설비 도입
  • ✔ 전자저울 자동계량 기능 도입
